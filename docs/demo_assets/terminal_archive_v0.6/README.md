# Archived terminal frames (v0.6.x era)

These terminal-scene PNGs were generated by
`docs/demo_assets/scripts/terminal_scene_renderer.py` during the
v0.6.x release window. They contain stale counts (45 native + 25 SIFT
= 70 tools, v0.6.1 boot banner). They were superseded in v0.7.1 when
the renderer was updated to emit 47 + 25 = 72 tools, v0.7.1 banner.

Replacement frames live in `../terminal/`.

Kept here as historical reference. Not used by any build pipeline.
