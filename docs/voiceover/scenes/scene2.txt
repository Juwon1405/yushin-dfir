One-line install on a clean SIFT Workstation. The script verifies Python, clones the repository, installs the dart underscore m c p package, and probes the SIFT toolchain — Volatility 3, MFTECmd, EvtxECmd, PECmd, RECmd, AmcacheParser, YARA, log2timeline, psort. Whichever binaries are on the system path get wired up automatically. Missing ones get clear environment-variable override hints. Then it verifies the full MCP tool surface is registered — native pure-Python plus SIFT Workstation adapters.
