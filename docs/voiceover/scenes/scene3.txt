Most agentic DFIR tools are a system prompt asking an LLM to behave well. We removed the ability to misbehave. The bypass test suite verifies that destructive operations — execute underscore shell, write underscore file, mount, eval — are not registered, and that path-traversal attempts against the evidence root are blocked at the MCP boundary itself. The SIFT adapter test suite verifies the same guarantees hold for all twenty-five subprocess wrappers. Adding tools never weakens the boundary, because the boundary is the canonical name set.
